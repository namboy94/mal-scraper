## Installation

### Via Pip (Preferred):

**As User (Preferred)**

    $ pip install malscraper --user
    
**Systemwide**

    # pip install malscraper
    or
    $ sudo pip install malscraper
    
### Via setup.py

**As User (Preferred)**

    $ python setup.py install --user
    
**Systemwide**

    # python setup.py install
    or
    $ sudo python setup.py install
